# OpenClaw / Bots Phenomenon — Research Notes

## What is OpenClaw?

Open-source autonomous AI agent (the lobster 🦞) that went viral in early 2026. Evolved from **Clawdbot → Moltbot → OpenClaw**. Created by Peter Steinberger.

- **Open-source autonomous AI agent** that hooks into LLMs (Claude, GPT, DeepSeek) and operates through messaging platforms (Signal, Telegram, Discord, WhatsApp)
- Runs locally, uses messaging services as its main UI
- Steinberger announced Feb 14, 2026 he's joining OpenAI; project moving to an open-source foundation

## Skills Ecosystem

- **5,400+ skills** in the official OpenClaw Skills Registry
- **36+ gaming-specific skills**, including:
  - **ClawVille** — persistent life simulation game for AI agents
  - **Dungeons and Lobsters** — bots-only fantasy campaigns played live
  - **Imitation Game Agent** — mandatory operational logic for playing
  - **Arena** — live AI app-building competitions with on-chain rewards

## The Bots Phenomenon

- **Moltbook** — Reddit-like online forum where users' OpenClaw agents post written content and interact with other chatbots through comments and upvotes/downvotes
- **Multiplayer game bots** — recent HN post about using OpenClaw to add playable bots to multiplayer games that "actually compete in-game instead of behaving like a script"
- **AI trading bots** — OpenClaw agents competing on live trading leaderboards (simul8or platform)
